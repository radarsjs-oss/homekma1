# homekma1

레이더 추적 기술 및 필터 알고리즘 분석 자료의 HTML 슬라이드 파일인 `tracking1.html`에 대한 링크를 정리합니다.

- Presentation (HTML slide): https://github.com/radarsjs-oss/homekma1/blob/3c74296a59c11b29714f62ee9e8bc749cf31d0d2/tracking1.html
- Raw HTML (원시 파일): https://raw.githubusercontent.com/radarsjs-oss/homekma1/3c74296a59c11b29714f62ee9e8bc749cf31d0d2/tracking1.html

로컬에서 확인하는 방법:

1. 파일을 클론하거나 다운로드합니다.
2. 파일이 있는 디렉터리에서 간단한 HTTP 서버를 실행합니다.

```bash
python3 -m http.server 8000
```

브라우저에서 http://localhost:8000/tracking1.html 를 열면 슬라이드를 확인할 수 있습니다.

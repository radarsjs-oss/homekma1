# homekma1

레이더 추적 기술 및 필터 알고리즘 분석 자료의 HTML 슬라이드 파일인 `tracking1.html`에 대한 링크와 외부에서 바로 열어볼 수 있는 접근 경로를 정리합니다.

- Presentation (GitHub view): https://github.com/radarsjs-oss/homekma1/blob/3c74296a59c11b29714f62ee9e8bc749cf31d0d2/tracking1.html
- Raw HTML (원시 파일): https://raw.githubusercontent.com/radarsjs-oss/homekma1/3c74296a59c11b29714f62ee9e8bc749cf31d0d2/tracking1.html

외부에서 바로 열어볼 수 있는 공개 미리보기(CDN/프록시) 링크:

- RawGitHack (빠른 미리보기): https://raw.githack.com/radarsjs-oss/homekma1/3c74296a59c11b29714f62ee9e8bc749cf31d0d2/tracking1.html
- jsDelivr CDN: https://cdn.jsdelivr.net/gh/radarsjs-oss/homekma1@3c74296a59c11b29714f62ee9e8bc749cf31d0d2/tracking1.html

GitHub Pages로 정식 호스팅하기 (추천):

1. 리포지토리 설정(Settings) > Pages로 이동합니다.
2. 배포(Deploy) 소스에서 `main` 브랜치의 `root`(또는 `docs/` 폴더)를 선택합니다.
3. Save 후 몇 분 기다리면 사이트가 배포됩니다.

배포가 완료되면 일반적으로 다음 URL로 접근 가능합니다:

- https://radarsjs-oss.github.io/homekma1/tracking1.html

(참고: GitHub Pages는 레포지토리 설정과 권한에 따라 약간의 시간이 걸릴 수 있으며, 퍼블릭 레포지토리여야 기본적으로 바로 외부에서 접근 가능합니다.)

로컬에서 확인하는 방법:

1. 파일을 클론하거나 다운로드합니다.
2. 파일이 있는 디렉터리에서 간단한 HTTP 서버를 실행합니다.

```bash
python3 -m http.server 8000
```

브라우저에서 http://localhost:8000/tracking1.html 를 열면 슬라이드를 확인할 수 있습니다.
